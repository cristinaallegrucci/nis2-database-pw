# nis2-database-pw
Pw basato sulla creazione di una base dati relazionale in ambito NIS2

## Contenuto

- Script SQL per la creazione dello schema
- Script di popolamento dati
- Trigger per la gestione dello storico (versioning)
- Query di interrogazione
- Schema relazionale e diagramma ER

## Struttura

- /sql → file SQL
- /docs → documentazione e diagrammi

## Obiettivo

Supportare la raccolta e gestione delle informazioni necessarie per la conformità alla normativa NIS2.
